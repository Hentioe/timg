使用 TIMG 将图片转为 ASCII 文本！


## Usage

Normal:

```` bash
./timg img.jpg
````

Output file: `img.jpg.html`

Limit size:

````
./timg -w 50 img.jpg
````

## Help:

```` bash
TIMG alpha
Hentioe Cl (绅士喵)
Image to ASCII

USAGE:
    timg [OPTIONS] <path>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -w, --width <width>    Set output ASCII text width

ARGS:
    <path>    The image file path
````
